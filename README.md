# Saheli — Your Personal Best Friend

A cute, soft, and emotionally supportive assistant for managing your day, capturing moments, and staying organized with love.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set `VITE_GEMINI_API_KEY` in `.env` to your Gemini API key
3. Run the app:
   `npm run dev`
